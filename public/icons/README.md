# PWA Icons

Place the following icon files in this directory:

| File                     | Size      | Purpose    |
|--------------------------|-----------|------------|
| `icon-192.png`           | 192×192   | any        |
| `icon-192-maskable.png`  | 192×192   | maskable   |
| `icon-512.png`           | 512×512   | any        |
| `icon-512-maskable.png`  | 512×512   | maskable   |

**Maskable icons** should have extra padding (~20%) around the logo so the
icon remains visible when the OS applies its own mask shape.

You can generate these from `neoLogo.jfif` using any image tool or an
online generator like <https://maskable.app/editor>.
